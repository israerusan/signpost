# Changelog

## 0.1.0

Initial release.

- Problem-first route map of community plugins, grouped into 9 goal-based categories.
- Live install status (enabled / installed / not installed) for every pick, read from the running app.
- "Open install page" deep-links to each plugin via the `obsidian://show-plugin` URI; GitHub fallback link.
- "Hide what I already have" toggle to focus on gaps.
- Collapsible sections that remember their state; fully bundled and offline; MIT-licensed.
